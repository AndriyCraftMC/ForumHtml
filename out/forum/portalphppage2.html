Документація API


API    Немає відповідей

kinash, 09-23-2022, 03:13 PM

Документація API форума: https://www.andriycraft.page/forum/api.php

Документація API сайта:

  • Добавити відгук: Метод: POST | Посилання: https://www.andriycraft.page/api/addreview | Параметри: username (нік на сервері), text (текст), email (пошта), stars (оцінка, від 0 до 5)
  • Получити список відгуків (JSON): Метод: GET | Посилання: https://www.andriycraft.page/api/getreviews
  • Проксі для картинок: Метод: GET  | Посилання: https://proxy.andriycraft.page/ | Параметри: url (посилання на картинку)
